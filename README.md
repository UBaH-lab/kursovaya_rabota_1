# Анализ транзакций

Приложение для анализа банковских транзакций из Excel-файла.

## Описание

Проект предоставляет функционал для:
- Генерации JSON-данных для веб-страниц
- Формирования отчетов
- Анализа транзакций

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
Установите зависимости:
bash

poetry install
Создайте файл .env на основе .env_template и укажите API-ключи
Использование
Запуск основного модуля:

bash

python src/main.py
Тестирование
Запуск тестов:

bash

pytest
Структура проекта

├── src/
│   ├── utils.py          # Вспомогательные функции
│   ├── main.py           # Точка входа
│   ├── views.py          # Функции для веб-страниц
│   ├── reports.py        # Функции для отчетов
│   └── services.py       # Сервисы
├── tests/                # Тесты
├── data/                 # Данные
├── .env                  # Переменные окружения
└── README.md
