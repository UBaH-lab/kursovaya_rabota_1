# Анализ транзакций

Приложение для анализа банковских транзакций из Excel-файла.

## Описание

Проект предоставляет функционал для:
- Генерации JSON-данных для веб-страниц (главная страница, страница событий)
- Формирования отчётов (траты по категориям, по дням недели, по рабочим/выходным)
- Анализа транзакций (поиск, выгодные категории, инвесткопилка)

## Возможности

### Веб-страницы (views.py)
- **Главная страница** — приветствие, карты, топ-5 транзакций, курсы валют, акции
- **Страница событий** — расходы/поступления по категориям за период (W/M/Y/ALL)

### Сервисы (services.py)
- **Выгодные категории** — анализ кешбэка по категориям
- **Инвесткопилка** — расчёт накоплений через округление трат
- **Простой поиск** — поиск транзакций по описанию/категориям
- **Поиск телефонов** — транзакции с номерами телефонов
- **Поиск переводов физлицам** — транзакции переводов частным лицам

### Отчёты (reports.py)
- **Траты по категории** — расходы по категории за 3 месяца
- **Траты по дням недели** — средние расходы по дням
- **Траты рабочий/выходной** — средние расходы в рабочие и выходные дни

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: poetry install
3. Создайте файл .env с API-ключами
4. Создайте user_settings.json с настройками валют и акций

## Использование

python src/main.py

## Тестирование

pytest --cov=src --cov-report=term-missing
